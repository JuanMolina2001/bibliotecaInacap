Antes de instalar la base de datos:

1. Instala XAMPP

Para comenzar, asegúrate de tener XAMPP instalado en tu sistema. Puedes descargarlo desde el siguiente enlace:

Descargar XAMPP

2. Abre XAMPP Control Panel

Una vez instalado XAMPP, abre el "XAMPP Control Panel" para gestionar los servicios. Verás una interfaz similar a la siguiente:



3. Inicia el servicio de MySQL

Dentro del "XAMPP Control Panel", asegúrate de que el servicio de MySQL esté marcado como "Start". Esto pondrá en marcha el servidor de base de datos MySQL:



4. Procede con la instalación de la ubicación predeterminada

Una vez que el servicio de MySQL esté en ejecución, puedes continuar con la instalación de la base de datos. La ubicación de instalación no es relevante para este proceso.
